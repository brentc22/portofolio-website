Automatic Parking System

Een slim systeem voor zelfrijdende parkeeracties zonder menselijke tussenkomst

Projectoverzicht

This system aims at replacing the conventional parking system with an IoT-based smart parking system by using RFID (radio-frequency identification). The users will be provided an entry card for getting access to the parking slot. The users will also be provided with an android based mobile application, using which they can know about the availability of the parking slot on their mobile phones. The users will need to maintain a minimum amount of balance in their entry card for getting access through the parking system, otherwise, the system will deny the access to enter. In this way, this smart parking system will help reduce human effort & time by using automation technology.

Hoe werkt het?

Het systeem bestaat uit een op schaal gemaakt voertuig dat is uitgerust met ultrasone sensoren om obstakels en afstanden te meten. Wanneer het voertuig wordt gestart, zoekt het automatisch naar een vrije ruimte. Zodra die wordt gedetecteerd, berekent het systeem de benodigde bewegingen om accuraat te parkeren.

Motoren worden aangestuurd via een H-brug, terwijl de sensordata realtime verwerkt worden door een microcontroller (zoals Arduino of ESP32). De status van het systeem wordt weergegeven via LED’s of een LCD-scherm.

Gebruikte Hardware & Software

  • Microcontroller: ESP32 of Arduino Uno
  • Sensoren: HC-SR04 ultrasoon sensoren, IR sensoren
  • Motoren: 2x DC motoren met wielen
  • Driver: L298N H-brug motor driver
  • Voeding: 12V batterij
  • Software: Arduino IDE / PlatformIO
  • Overige: Breadboard, draden, LEDs, LCD-scherm

Mogelijke Uitbreidingen

  • Gebruik maken van AI of machine learning voor herkenning van parkeerplaatsen
  • Live feedback via een webinterface of app
  • Inbouwen van camera’s voor visuele feedback
  • GPS-integratie voor parkeerlocatie tracking
